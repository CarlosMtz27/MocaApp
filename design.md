# MocaApp - Design System 💖

Este documento define la identidad visual de **MocaApp**, una aplicación diseñada para fortalecer los vínculos de pareja a través de la calidez, la sencillez y el romanticismo.

## 🌟 Filosofía de Diseño
MocaApp se basa en un enfoque **minimalista y cálido**. La interfaz debe sentirse como un refugio compartido: limpia para evitar el ruido cognitivo, pero con toques visuales suaves que transmitan afecto y cercanía.

**Principios clave:**
1.  **Suavidad:** Uso extensivo de bordes redondeados (`32.dp`) para eliminar la agresividad visual.
2.  **Espaciado:** Generoso uso de aire (padding) para resaltar lo que realmente importa: el contenido compartido.
3.  **Calidez:** Una paleta cromática basada en tonos rosados y crema que evocan amor y cuidado.

## ✍️ Tipografía
La tipografía busca un equilibrio entre la legibilidad moderna y un toque personal.

*   **Títulos (`Headline`):** Bold y directos para dar estructura.
*   **Cuerpo (`Body`):** Normal y con buen interlineado para facilitar la lectura de diarios y mensajes largos.
*   **Etiquetas (`Label`):** Tamaño reducido pero con suficiente contraste para información secundaria como fechas.

## 📱 Componentes y Estilo UI

### 1. Contenedores (Cards)
*   **Radio de esquina:** `24.dp` a `32.dp`.
*   **Elevación:** Mínima o inexistente (`tonalElevation`). Se prefiere el uso de colores de superficie sutiles sobre sombras pesadas.

### 2. Navegación
*   **Barra Inferior:** Flotante, con esquinas muy redondeadas y un fondo con ligera transparencia (`alpha = 0.95f`) para dar sensación de ligereza.
*   **Animación:** Transiciones suaves (Spring) al interactuar con iconos para una respuesta orgánica.

### 3. Iconografía
*   Se utilizan iconos redondeados (`Rounded`) de Material Design para mantener la coherencia con la suavidad de la interfaz.

## ✨ Experiencia de Usuario (UX)
MocaApp no es solo una herramienta, es un diario compartido. Cada interacción (enviar una nota, actualizar el estado de ánimo) debe sentirse gratificante y sencilla, eliminando barreras técnicas para priorizar la conexión emocional.
